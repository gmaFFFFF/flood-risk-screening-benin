Для объединения разнородных данных использован подход адаптивного пространственного взвешивания, учитывающий, что вес гидрологических свойств почв изменяется в зависимости от типа землепользования:


## Подходы в оценке водопроницаемости

### Экспертные оценки

>[!Info]
>Данный подход не применялся

Природные зоны с развитой растительностью значительно повышают водопроницаемость даже на малопроницаемых грунтах., однако не отменяют полностью влияние почв.  Веса распределяются: 70% Покрытие / 30% Грунт.

Влияние пашни равнозначно влиянию почв, т.к. почва регулярно подвергается механической обработке. Веса распределяются: 50% Покрытие / 50% Грунт.

Водопроницаемость территории полностью определяется  грунтами.

На урбанизированных и затапливаемых территориях 100% приоритет за покрытием. 

``` Python
Con("RasterCover" <= 2, ("RasterCover"*7 + "RasterSoil"*3) / 10, 
	Con("RasterCover" == 3, ("RasterCover" + "RasterSoil") / 2,
		Con("RasterCover" == 4, "RasterSoil",
			"RasterCover"
			)
		)
)
```
