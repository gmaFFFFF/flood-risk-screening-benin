#### Введение
Для анализа важен экстремальный объём осадков:
* Максимальная интенсивность дневных осадков [[Rx1day]], с заданным периодом повторяемости [[ARI]] — один мощный ливень.
* Максимальная интенсивность пятидневных осадков [[Rx5day]], с заданным периодом повторяемости [[ARI]] — несколько дней интенсивного дождя.

[[обеспеченность|Расчетная интенсивность]] [[Rx1day]] и [[Rx5day]] заданной обеспеченности ([[AEP]]) осуществляется на основе статистической оценки экстремумов (max) временного ряда осадков.

#### Источник данных
Готовые агрегированные данные из [humanData](https://data.humdata.org/dataset/ben-rainfall-subnational) не содержат нужных данных, поэтому потребовалось провести самостоятельную статистическую обработку журнала наблюдений за осадками, предоставленного [[CHIRPS Daily|CHIRPS]].

#### Выборка исходных данных
Выборка экстремальных (max) осадков осуществляется методом «[[Выборка максимума в блоке]]». 

> [!Info]
> Более современным и надежным методом является метод «[[Выборка превышающих порог]], но я его не осилил: нужно самостоятельно выбрать порог для ливня, сделать декластеризацию (отделить затяжной ливень от двух разных), а его математика  оказалась мне совсем не по силам.

Сделана гипотеза, что выборка соответствуют [[Распределение Гумбеля|распределению Гумбеля]].

Гипотеза проверена графически (доступно описание [[CHIRPS Daily#Графическая проверка соответствие выборки Распределение Гумбеля распределению Гумбеля|методики]]) путем сравнения стандартизованных исторических данных с теоретической кривой Гумбеля со стандартными параметрами:

#### Предсказание интенсивности дождя заданной обеспеченности 
Статистические данные, необходимые для расчета осадков заданной загружены с помощью [[CHIRPS Daily#Выгрузка данных с помощью google.Earth Engine Google Earth Engine ( google.Earth Engine GEE )|скрипта GEE]].

Загруженные данные представляют из себя многоканальный растр, который содержит статистики, необходимые для вычисления осадков заданной обеспеченностью по [[Распределение Гумбеля#Расчет Интенсивность события заданной обеспеченности интенсивности события заданной повторяемости|формуле]].

##### Шкала осадков
Для оценки потенциальной опасности осадков разработана унифицированная со  [[Шкала ущерба|шкалой ущерба]] 5-балльная шкала. Границы классов установлены на основе статистического анализа временных рядов [[Rx1day]] и [[Rx5day]] за $\approx$ 45-летний период на всей территории картографирования (экстент 1°з.д, 5°с.ш. — 6°в.д, 14°с.ш).

| Ежегодные осадки, мм | Минимум | Максимум | Среднее | СКО ($\sigma$) |
| -------------------- | ------- | -------- | ------- | -------------- |
| [[Rx1day]]           | 5.8     | 118.9    | 45.4    | 11.5           |
| [[Rx5day]]           | 11.9    | 262.5    | 90.4    | 18.8           |

Учитывая асимметрию распределения экстремальных осадков («тяжелые хвосты»), применена методика линеаризации квантилей через стандартное отклонение ($\sigma$) .

Границы 1-го класса установлен на отметке $0.5\sigma$ выше среднего. Ширина последующих интервалов составляет $2\sigma (\approx 23 мм)$ для суточных осадков и $2.5\sigma (\approx 47 мм)$ для 5-дневных сумм осадков.

Полученные пороговые значение округлены до удобных величин (50, 75, 100, …):

###### Переклассификация растра в [[ArcGIS]]
Используется функция растра
``` Python
Con(Rx1day <= 50, 1, 
    Con(Rx1day <= 75, 2, 
        Con(Rx1day <= 100, 3, 
            Con(Rx1day <= 120, 4, 5)
        )
    )
)
```

и

``` Python
Con(Rx5day <= 100, 1, 
    Con(Rx5day <= 150, 2, 
        Con(Rx5day <= 200, 3, 
            Con(Rx5day <= 240, 4, 5)
        )
    )
)
```
