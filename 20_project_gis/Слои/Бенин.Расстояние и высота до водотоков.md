
В качестве растра поверхности и стоимости относительно вертикального перемещения использована [[Бенин.ЦМР]].

## Предобработка исходных данных
### Водотоки

### [[Бенин.ЦМР]]
* [спроецирована](https://pro.arcgis.com/ru/pro-app/3.6/tool-reference/data-management/project-raster.htm) в СК [[Universal Transverse Mercator|UTM]] зона 31N.
* Гидрокоррекция:
    * [Заполнены](https://pro.arcgis.com/ru/pro-app/3.6/tool-reference/spatial-analyst/fill.htm) локальные понижения (ячейки без стока).    * 
* Сегментация
    * Площадь экстента расчета ($≈ 0,7 \: млн.\, км^2$) превышает территорию Республики Бенин для создания красивых иллюстраций. 
    * Из-за ограничений инструмента геообработки территория разбита на отдельные сегменты обработки по водосборным бассейнам.
        * Если использовать простые квадратные листы для обработки, то так как алгоритм не будет учитывать смежную территорию, то расчеты в смежных листах окажутся разными. Ошибка доходит до 30%. 
    * Крупные водосборы (площадью свыше $100\ тыс.\, км²$) дополнительно порезаны водотоками, т.к. иначе инструмент считает слишком долго и результат бывает неожиданным.
    

### [[ESA World Cover|World Cover]]
Карта  [переклассифицирована](https://pro.arcgis.com/ru/pro-app/3.6/tool-reference/spatial-analyst/reclass-by-ascii-file.htm) в шкалу шероховатости по Маннигу по данным [[Шероховатость водотоков#[Manning's Roughness Generator](https //github.com/mabdazzam/mannings_roughness_generator)|Manning's Roughness Generator]], но сверяясь с российскими справочными [[Шероховатость водотоков#Российские нормативы|таблицами]]

``` Python
remap_table = [  
[10, 0.094],  # Деревья  
[20, 0.066],  # Кустарники  
[30, 0.033],  # Луг  
[40, 0.035],  # Пашня  
[50, 0.060],  # Застройка (ВАЖНО: Исправлено с 0.018 на 0.06 для реалистичности)  
[60, 0.033],  # Грунт  
[70, 0.020],  # Снег  
[80, 0.035],  # Вода  
[90, 0.090],  # Затапливаемые  
[95, 0.225],  # Мангры  
[100, 0.060]  # Мхи, лишайники  
]
```

Застроенные земли включают как асфальт, по которому вода распространяется практически не встречая сопротивления, так и заборы, здания, которые создают серьезные препятствия. Значение коэффициента для застройки увеличено.

## Совместная классификация расстояния и высоты до водотока
>[!Tip]
>Совместная классификация не выполнялась

Два фактора взаимно дополняют друг друг, поэтому их можно попробовать учитывать в комплексе, а не по отдельности, балансируя весами.

Комбинирование факторов осуществлено с помощью инструмента [Нечеткое наложение](https://pro.arcgis.com/ru/pro-app/3.6/tool-reference/spatial-analyst/fuzzy-overlay.htm). Подходят два тип наложения «Product» (произведение) и «AND» (минимальный), т.к. чтобы случилось затопления должны действовать оба фактора, а не один из них.

Конечный выбор остановил на Product, т.к. комбинация нескольких факторов менее важна, чем каждого в отдельности.
