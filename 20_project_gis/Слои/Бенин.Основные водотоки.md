Информация о водотоках полученная из [HydroATLAS](https://www.hydrosheds.org/hydroatlas), выборка с уровнем детализации 10 оказалась непригодна для дальнейшей работы

Выполнялась попытка гармонизировать геометрическое положение водотоков из [HydroATLAS](https://www.hydrosheds.org/hydroatlas) с тальвегами на [[ЦМР]] методом [[ЦМР.Прожиг водотоков|«прожига»]]. Результат отрицательный.

Водотоки рассчитаны на основе откорректированной [[Бенин.ЦМР]] по методике [[Основные водотоки]]. 

Порог накопленного стока, определяющий принадлежность ячейки к гидросети, не имеет универсального теоретического значения и подбирается для каждой территории индивидуально. 

В настоящем исследовании пространство допустимых значений исследовалось по ряду чисел Маркова — разреженной последовательности, равномерно покрывающей несколько порядков величин по логарифмической шкале, что позволяет эффективно сузить диапазон поиска без перебора всех целых чисел подряд. 

Из рассмотренных кандидатных значений выбиралось то, при котором топология извлечённой гидросети наилучшим образом соответствовала видимым водотокам на картографической подложке. Принятое значение 7561 обеспечивает адекватное представление основных водотоков Республики Бенин при разрешении используемой ЦМР.
