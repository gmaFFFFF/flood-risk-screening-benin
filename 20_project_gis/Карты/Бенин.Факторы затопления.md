# Интегральная карта

## Название
Интегральная оценка интенсивности факторов затопления Республики Бенин

## Описание
Интенсивность факторов, потенциально приводящих к затоплению территории. Карта не содержит данных о глубине и вероятности затопления (риск затопления).

## Требования
[[Бенин.Базовые требования к картам|Базовые требования к картам]]

## Содержание карты
### Интегральная оценка интенсивности факторов затопления (восприимчивость территории к факторам затопления)
#### Название слоя: 
Восприимчивость территории к факторам затопления
#### Данные
Результат анализа «[[Факторы затопления|Факторы затопления]]».
> [!Info]
> Попарное сравнение выдумал [[AI]] [[Gemini]]. В принципе это можно даже обыграть в тексте

Веса факторов определены по методике [[Метод анализа иерархий. Алгоритм|Анализ иерархий]] затопления принять по [[АИ. Расчет весов критериев.xlsx]]:

| Критерий                                 | Вес критерия |
| ---------------------------------------- | ------------ |
| Выявленные зоны затопления               | 0.475936057  |
| Локальные понижения рельефа              | 0.171695626  |
| Крутизна склона                          | 0.100651698  |
| Осадки (интенсивность стока ливня)       | 0.098895307  |
| Проницаемость поверхности земли (SCS CN) | 0.074374331  |
| Удаленность от водотоков                 | 0.042365813  |
| Высота над ближайшим дренажным каналом   | 0.036081168  |

Коэффициенте согласованности (Consistency Ratio) $CR=0.09$, следовательно вышеуказанные веса можно считать согласованными. 
Максимальная несогласованность наблюдается в следующих парах:

| Критерий 1                  | Критерий 2                             | Суждение                                                   |
| --------------------------- | -------------------------------------- | ---------------------------------------------------------- |
| Выявленные зоны затопления  | Локальные понижения рельефа            | Несоизмеримо более значимый (9)                            |
| Локальные понижения рельефа | Высота над ближайшим дренажным каналом | Между «Равнозначны (1)» и «Немного более значимый (3)» (2) |
| Локальные понижения рельефа | Крутизна склона                        | Немного более значимый (3)                                 |

### Примечание
Карта не содержит данных о глубине и вероятности затопления (риск затопления).
