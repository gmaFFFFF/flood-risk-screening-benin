- ! Математическая формула метода взвешенного линейного наложения:
    - $S = \sum_{i=1}^{n} w_i x_i$, где $S$ — итоговый балл (индекс), $w_i$ — нормированный вес фактора $i$, $x_i$ — нормализованное значение фактора $i$.

- ! Ключевые предположения (допущения) метода:
    - Линейность: вклад фактора в общую оценку прямо пропорционален его значению.
    - Независимость факторов: факторы не должны описывать одно и то же явление во избежание двойного счёта (мультиколлинеарности).
    - Компенсируемость факторов: недостаток одного качества может быть восполнен избытком другого.

- ! Суть свойства компенсаторности:
    - Почему высокий балл по одному фактору перекрывает низкий по другому.
    - В аддитивной модели (каковой является @wlc) низкое значение @risk по одному параметру может быть арифметически скомпенсировано высоким значением по другому параметру.

- ! Область применимости свойства:
    - Допустимо при оценке интегрального потенциала (например, при геоинформационном скрининге на макроуровне).
    - Недопустимо, если хотя бы один из факторов является строгим лимитирующим барьером (например, если территория физически ограждена дамбой, никакой уровень осадков не должен компенсировать этот нулевой @risk).
  
- >: Вывод
  > Свойство компенсаторности делает @wlc отличным инструментом для скрининга, но требует осторожности при наличии строгих ограничений.

- >: Мост
  > Ограничения метода @wlc проявляются не только в компенсаторности, но и в типах обрабатываемых данных, в частности, категориальных.
