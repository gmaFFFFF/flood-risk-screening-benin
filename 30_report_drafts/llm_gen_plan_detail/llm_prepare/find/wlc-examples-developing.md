- Дата обращения: 2024-05-21
- Краткое содержание:
    - Примеры использования WLC (Weighted Linear Combination) для зонирования рисков затопления в развивающихся странах (Нигерия, Эфиопия, Саудовская Аравия, Камерун).
    - В Абеокуте (Нигерия) WLC применялся для агрегации факторов: топография (уклон/высота), интенсивность осадков, типы почв, плотность дренажа и использование земель. Факторы переклассифицировались по шкале 1–5.
    - В Эфиопии (Амбо) WLC позволил объединить карты опасностей (физические факторы) и карты риска (социально-экономические факторы: плотность населения).
    - Основные преимущества WLC для этих регионов: гибкость при дефиците данных, возможность интеграции качественных экспертных оценок (через МАИ) и количественных спутниковых данных, наглядность результатов для лиц, принимающих решения.
