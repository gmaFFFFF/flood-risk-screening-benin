### Краткое содержание:
Инструмент Distance Accumulation (Накопление расстояния) в ArcGIS Pro вычисляет стоимость перемещения между ячейками, учитывая расстояние по поверхности, растр стоимости, вертикальный и горизонтальный факторы.

#### Основная формула стоимости:
Cost = Surface Distance * Cost Surface Value * Vertical Factor * Horizontal Factor

#### Вертикальный фактор (VF):
VF является множителем, который учитывает дополнительные усилия для преодоления уклонов. Он рассчитывается на основе вертикального относительного угла перемещения (VRMA).
VRMA = arctan((z_to - z_from) / Surface Distance)

#### Типы функций вертикального фактора:
- Binary (Бинарная): VF=1 в диапазоне углов, иначе бесконечность.
- Linear (Линейная): VF = 1 + (Slope * VRMA).
- Symmetric Linear (Симметричная линейная): VF = 1 + (Slope * |VRMA|).
- Inverse Linear (Обратная линейная): VF = 1 - (Slope * VRMA).
- Tobler’s Hiking Function: Моделирует время пешего перехода.

#### Параметры:
- Zero Factor: Значение VF при VRMA = 0 (обычно 1.0).
- Cut Angles (Углы отсечки): Пороги VRMA, за которыми движение невозможно.
- Slope: Скорость изменения для линейных функций.

Инструмент позволяет моделировать гидравлическую удаленность, учитывая энергетические потери на трение и гравитацию.
