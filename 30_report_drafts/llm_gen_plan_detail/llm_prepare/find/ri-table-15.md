# Random Index (RI) table for AHP
URL: Various (including Saaty, 2008)
Date: 2024-05-XX

The Random Index (RI) values for the Analytic Hierarchy Process (AHP) for matrices of order $n=1$ to $15$.

| $n$ | Random Index (RI) |
| :--- | :--- |
| 1 | 0.00 |
| 2 | 0.00 |
| 3 | 0.58 |
| 4 | 0.90 |
| 5 | 1.12 |
| 6 | 1.24 |
| 7 | 1.32 |
| 8 | 1.41 |
| 9 | 1.45 |
| 10 | 1.49 |
| 11 | 1.51 |
| 12 | 1.53 |
| 13 | 1.56 |
| 14 | 1.57 |
| 15 | 1.59 |

The RI is used to calculate the Consistency Ratio ($CR = CI / RI$), where $CI$ is the Consistency Index. A $CR \leq 0.10$ is generally considered acceptable.
