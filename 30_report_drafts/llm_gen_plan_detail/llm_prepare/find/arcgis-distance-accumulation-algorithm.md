# Distance Accumulation Algorithm (Esri)

- **URL:** https://pro.arcgis.com/ru/pro-app/3.6/tool-reference/spatial-analyst/distance-accumulation-algorithm.htm
- **Дата обращения:** 13.05.2024
- **Краткое содержание:**
    - Вертикальный фактор (VF) используется для корректировки стоимости перемещения в зависимости от вертикального движения (уклона).
    - VF является множителем в формуле эффективного уклона: `C * S * M * HF * VF`.
    - Основным (и единственным явным) параметром для VF является вертикальный относительный угол движения (VRMA), определяемый уклоном между ячейками.
    - В документации не указано, что VF может зависеть от других динамических параметров (например, гидравлического уклона потока $i$) непосредственно в процессе накопления.
    - Ограничение: необходимо согласование единиц измерения. Если VF имеет размерность (например, время/расстояние), то растр стоимости должен быть безразмерным весовым коэффициентом.
