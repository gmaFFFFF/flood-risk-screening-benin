# Эмпирические подтверждения Ia = 0.05S для тропической зоны (Западная Африка)

**URL:** https://www.mdpi.com/2073-4441/9/6/392 (Komi et al., 2017)
**Дата обращения:** 2026-04-13

## Краткое содержание
Исследование "Distributed Flood Hazard Assessment in the Mono River Basin (West Africa)" (Komi et al., 2017) подтверждает, что для условий Западной Африки стандартный коэффициент начальных потерь $\lambda = 0.20$ значительно завышает потери. Калибровка модели в бассейне реки Моно показала, что значение $\lambda = 0.05$ обеспечивает наилучшее соответствие между моделируемым и наблюдаемым стоком.

Основные выводы:
- Стандартное значение $\lambda = 0.2$ не подходит для тропических регионов с высокой интенсивностью осадков.
- В Западной Африке сток начинает формироваться гораздо раньше из-за специфики почв и интенсивности ливней.
- Использование $\lambda = 0.05$ критически важно для точности карт затопления.
- При изменении $\lambda$ необходимо соответствующим образом пересчитать значения CN (Curve Number).
