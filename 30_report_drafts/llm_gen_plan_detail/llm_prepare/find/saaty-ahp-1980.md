- Дата обращения: 2024-05-21
- Краткое содержание:
    - Первоисточник метода анализа иерархий (МАИ/AHP).
    - Описывает четырехэтапный процесс: иерархическое моделирование (цель, критерии, альтернативы), попарные сравнения, расчет приоритетов (собственные векторы) и синтез.
    - Вводит фундаментальную шкалу 1–9 для квантификации субъективных суждений.
    - Описывает индекс согласованности (Consistency Ratio, CR) для проверки логичности экспертных оценок.
    - МАИ является ключевым методом для определения весов факторов в WLC, обеспечивая прозрачность и математическую строгость выбора.
