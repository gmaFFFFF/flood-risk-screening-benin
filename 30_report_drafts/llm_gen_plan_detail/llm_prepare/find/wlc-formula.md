- Краткое содержание: Метод взвешенной линейной комбинации (Weighted Linear Combination, WLC) использует математическую формулу: S = \sum_{i=1}^{n} w_i x_i, где S — итоговый показатель пригодности/риска, w_i — вес i-го критерия (сумма весов равна 1), x_i — стандартизированное значение i-го критерия. При наличии жестких ограничений формула дополняется умножением на произведение бинарных масок ограничений c_j: S = (\sum w_i x_i) * \prod c_j. Метод является компенсаторным: низкое значение одного фактора может быть компенсировано высоким значением другого.
