# Сводная таблица факторов затопления (Западная Африка / Бенин)

Данный артефакт обобщает теоретические сведения о факторах, определяющих риск ливневого и паводкового затопления в регионе.

## 1. Природные факторы (физическая основа)

| Группа факторов | Фактор | Показатель (Индекс) | Механизм влияния | Критические пороги (Шкалы) |
| :--- | :--- | :--- | :--- | :--- |
| **Климатические** | Экстремальные ливни | [[Rx1day]] | Формируют мгновенный избыток влаги, превышающий инфильтрацию. Причина флэш-паводков. | >100-120 мм (Существенный/Экстремальный) |
| | Продолжительные осадки | [[Rx5day]] | Приводят к насыщению почв и выходу крупных рек (Веме, Моно) на пойму. | >200-240 мм (Существенный/Экстремальный) |
| **Морфометрические** | Локальные понижения | [[TWI]] | Аккумуляция стока в замкнутых впадинах и долинах. | TWI > 8 (Существенный), >12 (Экстремальный) |
| | Вертикальное положение | [[HAND]] | Определяет энергетический барьер и вероятность затопления от русла. | HAND < 1.5 м (Высокий риск), < 0.5 м (Пойма) |
| | Крутизна склона | Склон (Slope) | Высокая крутизна ускоряет формирование стока; малые уклоны (<1°) способствуют застою воды. | < 1° (Зона аккумуляции / застоя) |
| **Гидрологические** | Водопроницаемость | [[SCS-CN]] | Определяет долю осадков, превращающуюся в сток (CN 40-100). | CN > 75 (Высокий сток), CN > 85 (Экстремальный) |
| | Гидрологическая группа почв | HSG (A-D) | Почвы группы D (глины) имеют минимальную инфильтрацию и максимальный риск стока. | Группа D (Максимальный риск) |

## 2. Антропогенные факторы (усилители риска)

| Фактор | Механизм влияния | Последствия в Западной Африке (Бенин) |
| :--- | :--- | :--- |
| **Урбанизация** | "Запечатывание" поверхности (бетонирование). | Рост коэффициента стока в городах (Котону, Абомей) в 1.5-2 раза. Сокращение времени добегания. |
| **Вырубка лесов** | Уничтожение лесной подстилки; создание термических контрастов. | Удвоение частоты интенсивных гроз за 30 лет. Потеря буферной емкости ландшафта. |
| **Застройка пойм** | Сокращение емкости естественных резервуаров (пойм). | Вытеснение паводковых вод на застроенные территории. Риск для жизни населения. |
| **Недостаток дренажа** | Засорение и малая пропускная способность каналов. | Хроническое затопление городских кварталов даже при штатных осадках (Rx1day < 50 мм). |

## 3. Синергетические эффекты

- **Порочный круг эрозии:** Вырубка лесов ведет к эрозии -> эрозия уменьшает глубину почвы -> уменьшается инфильтрационная емкость -> растет риск затопления.
- **Эффект наложения:** Сочетание морского прилива (в Котону) с речным паводком и интенсивным ливнем блокирует гравитационный дренаж.
