- ! Кривые интенсивности-продолжительности-повторяемости (IDF).
  - Понятие IDF-кривых как инструмента описания статистической структуры ливней.
  - Исходные данные: дискретные значения интенсивности для стандартных периодов повторяемости (ARI) и длительностей.

- ! Кусочно-логарифмическая интерполяция вероятностей.
  - Обоснование использования логарифмической шкалы для аппроксимации редких событий.
  - Формулы интерполяции интенсивности для нестандартных периодов повторяемости (например, ARI = 70 лет) на основе двух ближайших опорных точек.

- ! Интерполяция интенсивности во времени.
  - Понятие скорости затухания интенсивности дождя.
  - Математический вывод системы нелинейных уравнений для нахождения параметров аппроксимации внутри имеющихся временных диапазонов.
  - Использование логарифмических зависимостей для описания формы гиетограммы расчетного ливня.

- ! Определение расчетной продолжительности ливня.
  - Концепция времени добегания ($\tau_б$) как критического фактора формирования пикового стока.
  - Выбор расчетной продолжительности ливня, близкой к $\tau_б$ для конкретного водосборного бассейна.
