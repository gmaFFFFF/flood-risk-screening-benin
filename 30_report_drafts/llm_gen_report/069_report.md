#### Оценка согласованности {#sec:132}

Для верификации математической достоверности полученных весовых коэффициентов проведена оценка согласованности матрицы попарного сравнения. Данная процедура позволяет подтвердить непротиворечивость экспертных суждений и обоснованность их использования в многокритериальном анализе.


Для проверки логической согласованности экспертных суждений рассчитано отношение согласованности ($CR$) по формуле:
$$CR = \frac{CI}{RI}$$ {#eq:ahp_consistency}

Где:
- $CI$ — индекс согласованности (для данной матрицы $CI = 0,124$);
- $RI$ — случайный индекс для матрицы седьмого порядка ($RI = 1,32$).


Итоговое значение коэффициента согласованности (Consistency Ratio) $CR \approx 0,097$ свидетельствует о приемлемой согласованности суждений [@saaty1990]. Данный результат подтверждает, что вероятность случайного возникновения зафиксированных отклонений в матрице составляет менее 10%, что является научно обоснованным критерием качества экспертных оценок.

Несмотря на общую согласованность, был проведён анализ пар критериев с наибольшим отклонением от идеальной согласованности для оценки устойчивости модели (таблица -@tbl:inconsistency_pairs).

: Пары критериев с максимальной несогласованностью {#tbl:inconsistency_pairs}

| Критерий 1 | Критерий 2 | Суждение |
| :--- | :--- | :--- |
| Выявленные зоны затопления (JRC) | Локальные понижения рельефа (TWI) | Несоизмеримо более значимый (9) |
| Локальные понижения рельефа (TWI) | Высота над ближайшим дренажным каналом (HAND) | Между «Равнозначны (1)» и «Немного более значимый (3)» (2) |
| Локальные понижения рельефа (TWI) | Крутизна склона | Немного более значимый (3) |

Анализ данных таблицы -@tbl:inconsistency_pairs показывает, что основные отклонения связаны с оценкой влияния микрорельефа и локальных понижений, что объясняется сложностью разделения вклада данных факторов в формирование зон аккумуляции стока. Тем не менее, выявленные отклонения не выводят итоговый коэффициент $CR$ за критический порог, что позволяет признать матрицу пригодной для дальнейших вычислений.

#### Расчет весовых коэффициентов {#sec:133}

Вычисление итоговых весовых коэффициентов факторов затопления проводилось с использованием надстройки PowerQuery Microsoft Excel, что обеспечило прозрачность и воспроизводимость расчётов. Алгоритм реализации метода анализа иерархий (AHP) включал денормализацию матрицы попарного сравнения и применение L1-нормализации вектора приоритетов. Реализация расчётного алгоритма представлена в листинге -@lst:ahp_powerquery.

```{#lst:ahp_powerquery .powerquery caption="Реализация алгоритма AHP в PowerQuery"}
let
    Источник = ПопарноеСравнение,
    СуммаКоэффициентов = Table.Group(Источник, {"Критерий Правый"}, {{"Сумм", each List.Sum([Коэффициент]), type number}}),
    ДобавитьСумму = Table.NestedJoin(Источник,{"Критерий Правый"},СуммаКоэффициентов,{"Критерий Правый"},"СуммаКоэффициентов",JoinKind.Inner),
    РазвернутьСумму = Table.ExpandTableColumn(ДобавитьСумму, "СуммаКоэффициентов", {"Сумм"}, {"Сумм"}),
    Нормализовать = Table.ReplaceValue(РазвернутьСумму,
                                        each [Коэффициент],
                                        each [Коэффициент]/[Сумм],
                                        Replacer.ReplaceValue,
                                        {"Коэффициент"}),
    #"Удаленные столбцы" = Table.RemoveColumns(Нормализовать,{"Сумм"}),
    #"Сгруппированные строки" = Table.Group(#"Удаленные столбцы", {"Критерий Левый"}, {{"Вес критерия", each List.Average([Коэффициент]), type number}}),
    #"Переименованные столбцы" = Table.RenameColumns(#"Сгруппированные строки",{{"Критерий Левый", "Критерий"}})
in
    #"Переименованные столбцы"
```

Код листинга -@lst:ahp_powerquery реализует классическую процедуру нормализации столбцов и усреднения строк, что минимизирует вычислительную погрешность при определении весов факторов в рамках иерархической модели.

В результате выполнения алгоритма получены нормированные весовые коэффициенты для семи факторов затопления (таблица -@tbl:weight_results).

: Итоговые весовые коэффициенты факторов затопления {#tbl:weight_results}

| Критерий | Вес критерия |
| :--- | :--- |
| История (JRC) | 0,4759 |
| Топографический индекс влажности (TWI) | 0,1717 |
| Крутизна склона | 0,1007 |
| Сток ливня | 0,0989 |
| Водопроницаемость (SCS-CN) | 0,0744 |
| Удаленность от водотоков | 0,0424 |
| Вертикальная высота над водотоком (HAND) | 0,0361 |

Как следует из данных таблицы -@tbl:weight_results, фактор истории затоплений сохраняет доминирующее положение (почти 48% влияния), в то время как совокупный вклад морфометрических параметров рельефа (TWI и склон) составляет около 27%. Такое распределение приоритетов обеспечивает высокую достоверность скрининга в зонах исторических наводнений при сохранении чувствительности модели к новым потенциально опасным участкам.

Установленные весовые коэффициенты позволяют перейти к нормализации факторов для приведения их к сопоставимому виду перед итоговой агрегацией методом взвешенного линейного наложения.
