#### Архив CHIRPS Daily {#sec:92}

В качестве основного источника исторических сведений об осадках в исследовании используется набор данных Climate Hazards Center InfraRed Precipitation with Station data (CHIRPS Daily). Данный архив характеризуется высоким пространственным разрешением, составляющим 0,05° (примерно 5×5 км), и предоставляет непрерывный ряд ежедневных наблюдений, начиная с 1981 года. Наличие временного ряда продолжительностью около 45 лет делает CHIRPS Daily оптимальным фундаментом для выявления климатических трендов и анализа экстремальных явлений в рамках последующего моделирования (раздел -@sec:102).

В качестве альтернативного источника рассматривался продукт PERSIANN. Однако итоговый выбор был сделан в пользу CHIRPS Daily ввиду его лучшего соответствия специфическим задачам настоящего исследования, в первую очередь — благодаря большей продолжительности непрерывного ряда наблюдений, необходимого для статистически достоверной оценки экстремумов.

#### Кривые IDF (по данным XRain) {#sec:93}

Временное разрешение архива CHIRPS (одни сутки) достаточно для климатического анализа, однако оно является недостаточным для субчасового расчета стока (раздел -@sec:113) и гидродинамической симуляции (раздел -@sec:146). Для корректного воспроизведения динамики конкретного дождевого события требуются кривые интенсивности-продолжительности-повторяемости (IDF).

Источником субчасовых данных об интенсивности ливней выступает база геоданных XRain [@xrain2024], на основе которой осуществляется выгрузка параметров дождевых событий для заданного ряда периодов повторяемости (ARI). Следует констатировать, что альтернативные открытые источники субчасовых данных, обладающие сопоставимым пространственным разрешением и доступностью для исследуемой территории, в настоящее время отсутствуют.

#### JRC Global river flood hazard maps (LISFLOOD-FP) {#sec:94}

Одной из ключевых проблем моделирования риска затопления в исследуемом регионе является острый дефицит натурных (исторических) данных о границах и глубинах реальных наводнений. Для преодоления этого ограничения используются модельные данные продукта JRC Global river flood hazard maps [@baugh2024].

Данные глобальные карты опасности речных наводнений получены путем гидродинамического моделирования (LISFLOOD-FP) с пространственным разрешением около 90 м для различных сценариев повторяемости (ARI = 10–500 лет). В условиях недостатка фактических наблюдений применение глобальной карты опасности для периода повторяемости 100 лет рассматривается как строгий научный компромисс: эти модельные данные выступают в качестве прокси-переменной, заменяющей информацию об исторических затоплениях при верификации (раздел -@sec:126) и региональном скрининге.

Несмотря на то, что данные JRC не учитывают особенности локальной защитной инфраструктуры (дамбы, каналы), их использование признано допустимым исключительно для целей регионального геоинформационного скрининга. Альтернативные глобальные продукты сопоставимого качества и уровня детализации для рассматриваемой территории не выявлены.
