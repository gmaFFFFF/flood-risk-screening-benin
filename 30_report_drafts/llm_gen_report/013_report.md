### Математический аппарат метода анализа иерархий {#sec:ahp_math}

Метод анализа иерархий (МАИ) используется для определения весовых коэффициентов факторов затопления.

#### Шкала относительной важности Саати

МАИ основан на преобразовании субъективной оценки (в лингвистических единицах) в баллы от одного до девяти (таблица -@tbl:saaty_scale).
Для обратной зависимости используются величины $1/a_{i,j}$.

: Шкала относительной важности Саати {#tbl:saaty_scale}

| Интенсивность | Определение | Объяснение |
|---------------| --- | --- |
| 1             | Равнозначны | Два действия вносят одинаковый вклад в достижение цели |
| 3             | Умеренно превосходит | Опыт и суждение дают легкое предпочтение одному действию перед другим |
| 5             | Существенно превосходит | Опыт и суждение дают сильное предпочтение одному действию перед другим |
| 7             | Значительно превосходит | Одно действие сильно предпочитается другому; его превосходство подтверждается на практике |
| 9             | Абсолютно превосходит | Свидетельства в пользу предпочтения одного действия другому убедительны |
| 2, 4, 6, 8    | Промежуточные значения | Применяются для компромисса между двумя соседними суждениями |

Использование данной шкалы позволяет экспертам формализовать качественные различия между факторами, переводя субъективные предпочтения в дискретную числовую систему, пригодную для последующей математической обработки в рамках матриц попарных сравнений.

#### Матрица попарного сравнения

Сравнение критериев формализуется в виде квадратной матрицы $A = [a_{i,j}]$ размерностью $n \times n$. Она обладает свойствами положительной определенности ($a_{i,j} > 0$), обратной симметричности ($a_{j,i} = 1/a_{i,j}$) и единичной главной диагонали.

#### Расчет векторов приоритетов

Весовые коэффициенты находятся методом среднего геометрического по строкам с последующей нормализацией. Ненормализованный вес строки $i$ рассчитывается по формуле (-@eq:geom_mean):

$$ v_i = \sqrt[n]{\prod_{j=1}^{n} a_{i,j}} $$ {#eq:geom_mean}

где:
- $v_i$ — ненормализованный вес $i$-го критерия;
- $a_{i,j}$ — элемент матрицы попарных сравнений;
- $n$ — количество критериев.

Итоговые веса определяются через L1-нормализацию (-@eq:l1_norm):

$$ w_i = \frac{v_i}{\sum_{k=1}^{n} v_k} $$ {#eq:l1_norm}

Сумма весов вектора приоритетов $W$ равна единице, что необходимо для агрегации факторов в ГИС.

#### Оценка логической согласованности суждений

МАИ позволяет оценивать транзитивность экспертных суждений. Для выявления противоречий используется индекс согласованности (CI):

$$ CI = \frac{\lambda_{max} - n}{n - 1} $$ {#eq:consistency_index}

где:
- $\lambda_{max}$ — максимальное собственное число матрицы. 

Качество суждений оценивается сравнением $CI$ со значением случайной согласованности (Random Index, RI) из таблицы -@tbl:ri_table.

: Значения индекса случайной согласованности (RI) {#tbl:ri_table}

| $n$ | RI |
|-----|---|
| 1   | $0{,}00$ |
| 2   | $0{,}00$ |
| 3   | $0{,}58$ |
| 4   | $0{,}90$ |
| 5   | $1{,}12$ |
| 6   | $1{,}24$ |
| 7   | $1{,}32$ |
| 8   | $1{,}41$ |
| 9   | $1{,}45$ |
| 10  | $1{,}49$ |
| 11  | $1{,}51$ |
| 12  | $1{,}53$ |
| 13  | $1{,}56$ |
| 14  | $1{,}57$ |
| 15  | $1{,}59$ |

Коэффициент согласованности (CR) определяется по формуле (-@eq:consistency_ratio):

$$ CR = \frac{CI}{RI} $$ {#eq:consistency_ratio}

Суждения признаются допустимыми, если $CR \le 0{,}1$. Контроль этого порога гарантирует логическую устойчивость модели. Математический аппарат МАИ обеспечивает переход от субъективных знаний к объективным весам для пространственного анализа.
