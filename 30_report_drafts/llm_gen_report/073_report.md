### Выделение зон наибольшего риска затопления {#sec:139}

Для выделения зон сначала необходимо построить карту риска затопления. Необходимо отметить, что согласно методологии указанная карта на этапе скрининга является индексной и не содержит данные о физической глубине (в метрах) и точной вероятности (ARI) затопления. Основное назначение полученного результата — относительная пространственная дифференциация территорий по уровню угрозы для целей приоритизации.

#### Определение весов факторов риска затопления {#sec:140}

Для оценки риска используется двухфакторная модель, сопоставляющая физическую опасность (Hazard) и подверженность (Exposure). Поскольку модель включает всего два фактора, классический метод анализа иерархий (AHP) вырождается в прямое экспертное суждение, поэтому веса факторов определены прямым экспертным сравнением автора.

Плотность населения (Population Density) принята как доминирующий фактор при оценке риска с весом 0,65. Данное решение обосновано приоритетностью минимизации угроз жизни и здоровью людей (социально-экономическая значимость ущерба) по сравнению с оценкой физической интенсивности процессов на незаселённых территориях. Итоговые веса факторов представлены в таблице -@tbl:risk_factors_weights.

: Весовые коэффициенты факторов риска затопления {#tbl:risk_factors_weights}

| Фактор | Вес фактора |
| :--- | :--- |
| Плотность населения (Exposure) | 0,65 |
| Интегральная оценка интенсивности факторов (Hazard) | 0,35 |

Из данных таблицы -@tbl:risk_factors_weights следует, что итоговая карта риска в значительной степени определяется демографическим давлением. Такой перекос весов позволяет отфильтровать обширные незаселенные поймы и сфокусировать внимание на урбанизированных территориях, где социально-экономические последствия наводнений максимальны.

#### Нормализация данных {#sec:141}

Процедура нормализации данных (Data Normalization) реализована в среде ArcGIS Pro с помощью специализированного инструмента Suitability Modeler. Процесс основан на принципах нечеткой логики (Fuzzy Logic), что позволяет трансформировать исходные данные в сопоставимую балльную шкалу (1–5).

Интегральная карта факторов затопления (Hazard) была нормализована с помощью функции MSLarge (Mean Standard Deviation Large) с параметрами: средний множитель — 1, множитель СКО — 1. Это позволило выделить территории с аномально высокими значениями кумулятивной опасности относительно среднего уровня по стране. Параметры функции аппроксимации представлены на рисунке -@fig:hazard_approx.

![График аппроксимации функции принадлежности для интегральной карты факторов затопления (Hazard)](prepare/artifacts/hazard_approx.svg){#fig:hazard_approx}

Как показано на рисунке -@fig:hazard_approx, использование функции MSLarge позволяет сместить фокус на экстремальные значения интенсивности факторов, отсекая территории с фоновым уровнем опасности.

Для нормализации плотности населения (Exposure) использована функция Large, обеспечивающая экспоненциальный рост значимости фактора при высоких значениях плотности. Параметры функции: средняя точка — 36,0, разброс — 0,73. Граничные условия шкалы определены следующим образом: значения менее 1 чел/ячейку соответствуют баллу 1 (минимальный вклад в риск), значения более 250 чел/ячейку соответствуют баллу 5 (максимальный вклад).

Параметры функции аппроксимации для плотности населения представлены на рисунке -@fig:ghs_pop_approx.

![График аппроксимации функции принадлежности для плотности населения (GHS-POP)](../../input/img/ghs_pop_approx.png){#fig:ghs_pop_approx}

Анализ графика на рисунке -@fig:ghs_pop_approx подтверждает, что выбранная функция Large с заданной средней точкой (36,0) обеспечивает корректное выделение высокоурбанизированных зон как приоритетных объектов защиты при оценке риска.
