#### Моделирование гидрологической связности рельефа и водосборных бассейнов {#sec:58}

После завершения этапа подготовки поверхности цифровой модели рельефа (ЦМР) и исключения артефактов радарной съемки необходимо перейти к моделированию динамических процессов — маршрутизации поверхностного стока и формированию бассейновой структуры территории. Целью данного этапа является расчет направлений движения воды, определение границ водосборных бассейнов и выделение топологически согласованной гидрографической сети, что служит фундаментом для последующего расчета гидравлических индексов опасности затопления, таких как топографический индекс влажности (TWI) и высота над ближайшим водотоком (HAND).

##### Маршрутизация стока и выделение гидросети {#sec:59}

Первым шагом в построении гидрологической связности ландшафта является определение путей движения воды из каждой ячейки растра. Для решения этой задачи в рамках настоящего исследования применен алгоритм D8 (Determinism 8), согласно которому вода из центральной ячейки направляется в одну из восьми соседних ячеек, имеющую максимальный уклон поверхности. 

Геоинформационный расчет направления (Flow Direction) и накопления стока (Flow Accumulation) позволяет математически описать структуру водотоков. Выделение гидросети осуществляется на основе анализа растра накопления стока, где значение каждой ячейки соответствует количеству вышерасположенных ячеек, сток из которых проходит через данную точку.

Определение порога руслоформирования является ключевым параметром генерализации гидросети. Для территории Республики Бенин установлен порог в 7561 ячейку. Данное значение определяет минимальный объем накопленного стока, при превышении которого ячейка классифицируется как элемент русла. Для повышения физической достоверности выделения активных русел в расчетах использован весовой растр экстремальных осадков Rx5day, что позволяет учесть пространственную неравномерность климатической нагрузки (подробнее см. -@sec:48).

Для классификации полученных водотоков и определения их иерархической структуры применен метод Штралера (Strahler Stream Order) (рисунок -@fig:stream_v_concept). Векторизация гидросети (инструмент Stream to Feature) позволила получить линейные объекты, необходимые для последующего топологического анализа и визуализации основных водных артерий региона.

![Концептуальная схема иерархии водотоков по методу Штралера](prepare/artifacts/stream_v.svg){#fig:stream_v_concept}

Из рисунка -@fig:stream_v_concept видно, что метод Штралера позволяет формализовать иерархию гидрографической сети, где последовательное слияние притоков увеличивает порядок водотока, отражая процесс пространственной агрегации стока.

##### Расчет водосборных бассейнов {#sec:60}

На основе ранее рассчитанных направлений стока выполняется агрегация территории в замкнутые гидрологические единицы — водосборные бассейны. Автоматизированное выделение границ бассейнов осуществляется путем поиска совокупности ячеек, сток из которых направлен в общую точку выхода (mouth) или на границу расчетной области. Для этих целей используются специализированные инструменты, например, `ArcGIS Basin`.

Вычисленные бассейны требуется генерализовать с учетом масштаба исследования. 

Пространственное распределение водосборов отражает макрорельеф территории и формирует естественные границы для локализации зон возможного затопления.

Сформированная иерархия водосборов и топологически согласованная гидросеть позволяют перейти к расчету производных гидравлических индексов, которые детально рассматриваются в следующих разделах.
