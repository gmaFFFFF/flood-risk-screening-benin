#### Предварительная обработка {#sec:115}

На основе подготовленных исходных слоев землепользования и гидрологических групп почв выполняется построение расчетной карты номеров кривых стока (CN), являющейся фундаментом моделирования по методу SCS-CN.

Перекодировка пространственных данных ESA WorldCover и HiHydroSoil в значения кривых стока выполнена на основе соответствующих таблиц, извлеченных из руководства пользователя инструмента r.curvenumber [@grass_curvenumber]. При этом в качестве расчетных приняты условия предшествующего увлажнения AMC II (нормальные условия), которые являются стандартными для региональных оценок, обеспечивая баланс между засушливыми и переувлажненными периодами.

Обработка специфических классов растительности и покрытия потребовала адаптации классической методики. Классам мангров и снега/льда, для которых стандартные таблицы SCS-CN не предусматривают значений коэффициента CN, присвоено значение CN = 100, соответствующее абсолютно водонепроницаемой поверхности. Для мангровых лесов это решение физически обосновано постоянным водонасыщением почв в условиях приливно-отливного режима. Наличие класса снега и льда на территории тропической Западной Африки соответствует либо постоянному ледяному покрову (для которого водонепроницаемость является корректным допущением), либо артефакту классификации, не влияющему на итоговые результаты ввиду единичности таких пикселей. Кроме того, принятое значение CN = 100 исключает деление на ноль в расчётных формулах метода SCS-CN.

Результатом предварительной обработки является интегральная карта Curve Number для всей территории исследования, представленная на рисунке -@fig:scs-cn. Полученные значения в дальнейшем подвергаются нормализации (см. раздел -@sec:134).

![Карта распределения значений Curve Number (CN) на территорию Республики Бенин](../../input/img/scs-cn.jpg){#fig:scs-cn}

Анализ карты на рисунке -@fig:scs-cn выявляет пространственную неоднородность потенциала стока и его высокое абсолютное значение (свыше 50).
