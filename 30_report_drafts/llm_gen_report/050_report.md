#### OpenStreetMap (OSM) {#sec:95}

На этапе детального гидродинамического моделирования затопления в городских условиях (раздел -@sec:146) критически важно учитывать влияние искусственных объектов на растекание воды. Для создания пространственных масок водонепроницаемых поверхностей (контуры зданий, дороги, дренажные каналы) используются данные открытого картографического проекта OpenStreetMap (OSM) [@osm2025]. В расчетной гидродинамической схеме данные векторные объекты выступают в качестве непреодолимых барьеров или преимущественных путей стока.

Кроме того, картографические данные OSM используются в качестве подложки при оформлении итоговых отчетных карт среднего масштаба. В настоящее время альтернативные источники пространственных данных о городской инфраструктуре сопоставимого уровня детализации и обладающие свободной лицензией, допускающей их использование в научных исследованиях, отсутствуют.

#### Глобальные базовые карты (Natural Earth) {#sec:96}

Для целей мелкомасштабного картографирования, в частности при подготовке отчетных и обзорных карт регионального масштаба, используется набор геопространственных данных Natural Earth [@naturalearth2024]. Данный продукт предоставляет генерализованные векторные и растровые базовые слои, оптимально подходящие для визуализации результатов на уровне страны или крупных регионов. Аналогично данным OSM, выбор Natural Earth обусловлен высоким качеством генерализации и отсутствием сопоставимых альтернатив со свободной лицензией.

#### Информация о зданиях из БД Global Human Settlement Layer (GHS-BUILT-H) {#sec:97}

> [!warning]
> Симуляция не проводилась. Раздел содержит предварительные данные.

При построении цифровой модели городского рельефа для гидродинамической симуляции недостаточно только плановых контуров застройки; требуется информация о высотных отметках зданий, формирующих 3D-барьеры на пути водного потока. В качестве источника данных о высоте зданий выбран набор Global Human Settlement Layer - Building Height (GHS-BUILT-H) [@ghs_pop_doc].

В качестве альтернатив рассматривались такие глобальные продукты, как World Settlement Footprint (WSF) 3D [@wsf3d2024], Overture Maps [@overture2024] и Google Open Buildings 2.5D Temporal [@googleob25d2024]. Однако итоговый выбор был сделан в пользу GHS-BUILT-H благодаря его оптимальному пространственному разрешению, консистентности с другими слоями базы GHSL, используемыми в данном исследовании, а также открытой политике распространения данных.

#### Демографические данные Global Human Settlement Layer (GHS-POP) {#sec:98}

Оценка подверженности населения риску затопления требует точных данных о пространственном распределении жителей исследуемой территории. Для решения этой задачи используется растровый набор данных GHS-POP, который отображает плотность населения по регулярной сетке пространственных ячеек. Данный растр выступает в качестве базового фактора «Плотность населения» при выделении зон наибольшего риска (раздел -@sec:136).

В ходе исследования рассматривались альтернативные демографические продукты, такие как WorldPop и LandScan. Выбор растра GHS-POP аргументируется его высокой степенью совместимости с другими используемыми наборами данных GHSL (в частности, GHS-BUILT-H) и форматом представления данных, который обеспечивает бесшовную интеграцию в алгоритмы растрового геоинформационного анализа.
