## Алгоритм
1. Построить иерархию. Проблема декомпозируется на уровни: цель (на вершине), критерии (промежуточный уровень) и альтернативы (нижний уровень).
2. Составить исчерпывающий список критериев сравнения
3. ЛПР выполняет попарное сравнение элементов на одном уровне относительно их важности для вышестоящего элемента (например, сравнение Критерия А и Критерия Б по их влиянию на Цель), используя [[Метод анализа иерархий. Шкала сравнения критериев|шкалу сравнения]]: ![[Метод анализа иерархий. Шкала сравнения критериев|шкалу сравнения]] 
4. Составить [квадратную матрицу](https://ru.wikipedia.org/wiki/Квадратная_матрица) сравнения критериев, у которой все элементы [главной диагонали](https://ru.wikipedia.org/wiki/Главная_диагональ) равны единице. [[#Пример квадратной матрицы]].
5. Рассчитать веса критериев  (нормализованные оценки вектора приоритетов):
    * [[#Расчет весов критериев. Способ 1|Способ 1]]
    * [[#Расчет весов критериев. Способ 2|Способ 2]]
6. Проверка логической согласованности попарного сравнения
    * Для проверки используется коэффициент согласованности (Consistency Ratio, CR)
        * Если $CR \leq 0.1$, то данные согласованны
        * Если $CR > 0.1$, то данные **не** согласованны
    * Расчёт коэффициента согласованности
        * $CR = \frac{CI}{RI}$, где $CI$ — индекс согласованности, $RI$ — [[#Индекс случайной согласованности матрицы схожего порядка (Random Index, RI)]]
7. Поиск несогласованных пар сравнений
    * Построить [[#Матрица попарной согласованности|матрицу попарной согласованности]]
    * Выбрать 3 максимальных значения
8. Веса альтернатив по каждому критерию агрегируются с весами самих критериев для получения общего (глобального) приоритета для каждой альтернативы и выбора наилучшего варианта

## Расчет весов критериев. Способ 2
1. Рассчитать оценку компонентов собственного вектора (среднее геометрическое по строке $\sqrt[n]{a_{i,1} \cdot a_{i,2} \cdot … \cdot a_{i,n}}$)
2. [[Нормализация данных#«Нормализация к сумме 1» (L1-нормализация / приведение к долям / scale to unit sum)|«Нормализовать к сумме 1»]] столбец  оценки компонентов собственного вектора
    * Вычислить сумму
    * Разделить каждую оценку на сумму

## Индекс согласованности (Consistency Index, CI)
$$
CI = \frac{λ - n}{n-1}
$$
где $n$ — количество критериев,
$λ$ — главное собственное число (среднее $λ_i$ для каждого критерия):
$$
λ = \frac{\sum_{i}^{n}{λ_i}}{n}
$$
где
$i$ — номер строки,
$λ_i$ вычисляется по формуле:
$$
λ_i = \frac{SW_i}{w_i}
$$
где:
$w_i$ — вес критерия $i$,
$SW_i$ — взвешенная сумма для каждого критерия:
$$
SW_i = \sum_{1}^{n}{a_{i,j}w_j}
$$
где
$a_{i,j}$ — значение из исходной матрицы попарного сравнения, где _i_ означает строку, а _j_ - столбец
$w_j$ — вес соответствующего критерия

## Матрица попарной согласованности
Матрица строится из исходной матрицы по формуле:
$$
ε_{i,j}=\frac{a_{i,j}w_j}{w_i}
$$

### Индекс случайной согласованности матрицы схожего порядка (Random Index, RI)
RI — это предопределенное статистическое значение, которое получилась бы при случайном выборе количественных суждений. RI служит эталоном для вычисления CR при попарных сравнениях. Разработанные Саати[^saaty], значения RI получены на основе моделирования методом Монте-Карло для каждого размера матрицы. Полученные значения RI для каждого размера матрицы (количество критериев, n) приведены в таблице ниже.
