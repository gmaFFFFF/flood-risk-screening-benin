Информация о гидрологических свойствах почв по [[Hydrologic Soil Group]].
Документация на предыдущую версию [здесь](https://www.futurewater.nl/wp-content/uploads/2015/05/HiHydroSoil-A-high-resolution-soil-map-of-hydraulic-properties.pdf)

Географическое разрешение 1/480 десятичных градусов, что соответствует прогнозируемому разрешению приблизительно 250 м.


| Исходный код | Гидрологическая группа почв |
| ------------ | --------------------------- |
| 1            | A                           |
| 2            | B                           |
| 3            | C                           |
| 4            | D                           |
| 14           | A/D                         |
| 24           | B/D                         |
| 34           | C/D                         |

Продукт является альтернативой для [[HYSOGs250m]]
