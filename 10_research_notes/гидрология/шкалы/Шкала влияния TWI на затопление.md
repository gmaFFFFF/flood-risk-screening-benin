#### Переклассификация растра в [[ArcGIS]]
Используется инструмент один из инструментов группы Переклассификация ([Reclassify by ascii file](https://pro.arcgis.com/ru/pro-app/3.6/tool-reference/3d-analyst/reclass-by-ascii-file.htm))  или функция растра
``` Python
Con(twi <= 4, 1, 
    Con(twi <= 6, 2, 
        Con(twi <= 8, 3,
            Con(twi <= 12, 4, 5
) ) ) )
```
