Уровни шкалы соответствуют шкале [[Шкала ущерба|Шкале ущерба]].

| Код | Уровень воды от, м | Характеристика ущерба           |
| --- | ------------------ | ------------------------------- |
| 1   | 0,2                | [[Затопление]]                  |
| 2   | 0,3                | Сильное затопление              |
| 3   | 0,5                | Наводнение                      |
| 4   | 0,8                | Опасное наводнение              |
| 5   | 1,3                | Катастрофическое [[Наводнение]] |

#### Переклассификация растра в [[ArcGIS]]
Используется инструмент один из инструментов группы Переклассификация ([Reclassify](https://pro.arcgis.com/ru/pro-app/3.6/tool-reference/3d-analyst/reclassify.htm)) или функция растра
``` Python
Con(depth <= .2, 1, 
    Con(depth <= .3, 2, 
        Con(depth <= .5, 3, 
            Con(depth <= .8, 4, 5)
        )
    )
)
```
